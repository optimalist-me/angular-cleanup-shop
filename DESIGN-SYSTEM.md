# DESIGN-SYSTEM.md – Structural Cleaning Store

This document defines the visual and stylistic direction of the Angular Cleaning Service website/app.

It is binding for:

- Human designers
- Frontend developers
- AI agents (Copilot, ChatGPT)

This is not decoration.
This is controlled visual positioning.

---

# 1. Visual Philosophy

The style must feel:

- Calm
- Structured
- Confident
- Minimal
- Slightly playful

The goal is:

Professional credibility first.
Playful metaphor second.

Never the other way around.

---

# 2. Cartoon Style Guidelines

Cartoon style is used as a metaphor layer, not as entertainment.

## Illustration Principles

- Flat vector style
- Soft shadows (minimal)
- Clean outlines
- Consistent line thickness
- No exaggerated expressions
- No childish exaggeration

Products (e.g. Angular Degreaser) may have:

- Subtle visual humor
- Minimal facial expression (if any)
- Clean, geometric shapes

Avoid:

- Comic fonts
- Speech bubbles
- Over-saturated colors
- Visual noise
- Meme-like graphics

---

# 3. Color System

Primary palette:

- Neutral background (white / light grey)
- Dark slate / charcoal text
- 1 primary accent color
- 1 secondary accent color

Cartoon elements must use the same palette.
No rainbow palette.

Consistency > creativity.

---

# 4. Typography

Typography must feel:

- Modern
- Professional
- Clean

Avoid:

- Comic-style fonts
- Decorative headline fonts
- Overly condensed type

Hierarchy must be clear and calm.

---

# 5. Layout Rules

- Generous whitespace
- Clear visual separation between sections
- No heavy gradients
- No busy backgrounds
- Cards should feel structured and predictable

Minimal animations only.

---

# 6. UX Tone

Microcopy must be:

- Calm
- Slightly witty
- Never sarcastic
- Never childish
- Never ironic in a mocking way

Example:

Allowed:
"Removes architectural buildup from large Angular components."

Not allowed:
"Blasts your spaghetti code into oblivion!"

---

# 7. Architecture Mode UI

Architecture Mode must:

- Use subtle overlays
- Use muted highlight colors
- Never overwhelm normal users
- Keep panels small and dismissible

The site must remain usable when architecture mode is off.

---

# 8. What We Explicitly Avoid

- Over-animation
- Over-illustration
- Visual clutter
- Excessive icons
- Cartoon dominance over content

The credibility must always win.

---

Structural Cleaning Store  
Minimal. Clear. Playful with restraint.

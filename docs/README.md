## Angular Cleanup Playbook

This playbook is written using the latest Angular versions (v21+)
to reflect **current best practices**.

However:

> Applying this playbook does **not** require upgrading a client application to Angular v21.

Our engagements focus on:
- architecture
- component boundaries
- state management
- RxJS hygiene
- testing and guardrails

These improvements are usually applicable to:
- Angular 14+
- Angular 15–20
- sometimes even earlier versions (with adaptations)

Framework upgrades are **optional**, explicitly scoped, and never a prerequisite.

Core principles

  • Explicit domain boundaries and ownership.
  • Signals-first state and computed derivations.
  • Small, reviewable PRs that stay within a single domain when possible.

Modern Angular means

  • Typed reactive forms and explicit side effects.
  • RxJS hygiene: fewer scattered subscriptions, clearer async flows.
  • Pragmatic testing and predictable change detection.

Upgrade stance

Cleanup works within your current version. Upgrades are optional and scoped separately. Structural clarity makes them safer later.

Version assessment

  • Current Angular version and supported practices.
  • Upgrade feasibility and risk without assuming upgrades.
  • Constraints that impact cleanup sequencing.

Stabilize first

  • Identify entropy hotspots: PR sprawl, coupling hubs, and regression risk.
  • Agree on boundaries, ownership, and the ‘public surfaces’ between domains.
  • Define a safe sequencing plan: smallest high-leverage interventions first.

Component scope

  • Single responsibility and clear inputs/outputs.
  • Template complexity kept low.
  • Side effects live at the boundary, not scattered across components.

State ownership

  • Local signals by default; promote only when multiple consumers exist.
  • Commands for side effects; computed for derived state.

RxJS hygiene

  • Subscribe once (feature boundary), not everywhere.
  • Flattened operator chains and named pipeline steps.

Shared libraries

  • Shared is not a dumping ground.
  • Extract only with clear ownership and a minimal public API.

Recommended metrics

  • PR size stays reviewable (small change surface; fewer cross-domain touches).
  • No circular dependencies between domains.
  • Hotspots tracked: scattered subscriptions, large components/services, flaky tests.

Guardrails

  • Module boundary enforcement (Nx, ESLint rules, and CI checks — depending on your stack).
  • Gradual constraint tightening: block new violations, reduce legacy ones incrementally.
  • AI-assisted refactors are allowed only with governance: no secrets, no blind autopilot.

Cleanup sessions
